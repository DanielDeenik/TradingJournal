Import Instructions

The trades must be saved in CSV format and a max file size of 1MB.

The data format looks like the following:

date,symbol,num_shares,buy_price,sell_price,notes
2021-01-14,KPTI,5,452,448,
2021-01-27,INSG,310.23984,12.48,13.24,

A few notes about this data format:

  • Date: must be yyy-mm-dd form-data.
  • Data should be comma delimited.
  • The rows should be in the order shown.
